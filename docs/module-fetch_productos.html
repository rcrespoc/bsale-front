(inner) traerProductosPorCategoria(id, limiteopt, desdeopt) → {Promise.<Object>}

Función que recupera productos por categoria.
Parameters:
Name Type Attributes Default Description
id String Id de la categoria a la cual pertenecen los productos que se van a retornar.
limite Number <optional>
 6 La cantidad máxima de productos que se van a retornar por búsqueda.
desde Number <optional>
 0 Desde qué producto se va a iniciar la búsqueda. Sirve para paginación.
